# SweetUnlocker
Just a .bat menu to assist the sk-zk extractor.

[Dependencies](https://github.com/sk-zk/Extractor/releases)

To install, simply place the extractor.exe file in the same location as SweetUnlocker.bat

Download [here](https://github.com/SweetRlk/SweetUnlocker/releases) and extract the .zip file.
